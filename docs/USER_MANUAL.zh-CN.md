# TaffyQSL 用户手册

TaffyQSL 是一款用于签名业余无线电 QSO 日志并上传至 ARRL Logbook of The World (LoTW) 的 Android 应用程序。受 TrustedQSL 启发，提供了一个手机端的方式来管理证书、台站和 QSO 记录。

> ⚠️ 重要提示
>
> TaffyQSL 无法生成新的 LoTW 证书。
> 您必须使用 TrustedQSL (TQSL) 创建证书并将其导出为 `.p12` 文件，然后才能导入到本应用中。

---


## 安装

### 下载

TaffyQSL 可通过以下渠道获取：

- **GitHub Releases**：从 [Releases 页面](https://github.com/sophiel-meow/TaffyQSL/releases) 下载最新的 APK
- **F-Droid**：在 F-Droid 仓库中可用（即将推出）

### 安装步骤

1. 将 APK 文件下载到您的 Android 设备
2. 如果提示，请在设备设置中启用「允许安装未知来源的应用」
3. 打开 APK 文件并按照安装提示操作
4. 根据要求授予必要的权限（其实只有INTERNET权限 (⁄ ⁄•⁄ω⁄•⁄ ⁄)）

   - 不需要电话权限、位置权限，什么都不需要
   - 大部分 Android 版本可能无法让用户管理INTERNET权限——当然还是有办法关掉它的，只不过网络相关的功能就不能用了。
   - GrapheneOS的用户：有一个额外可管理的Sensors权限（加速度计等传感器），直接拒绝就好，本程序完全不会使用加速计来监听用户（笑）。

## 快速开始

### 导入证书

在签名 QSO 之前，您必须导入您的 LoTW 证书。

**重要说明：**
- TaffyQSL **不支持**生成新证书
- 您必须使用 **TQSL** (TrustedQSL) 将证书导出为 `.p12` 文件
- **密码是必需的**：由于 Android Keystore 的限制，从 TQSL 导出时必须设置密码（即使 TQSL 允许无密码导出，Android 也需要密码）

#### 导入步骤：

1. **从 TQSL 导出**（在您的计算机上）：
   - 打开 TQSL
   - 进入 "Callsign Certificates"（呼号证书）
   - 选择您的证书 → "Save the Callsign Certificate"（保存呼号证书）
   - 选择保存位置
   - **设置密码**（Android 必需）
   - 保存文件并传输到您的手机

2. **导入到 TaffyQSL**：
   - 打开 TaffyQSL
   - 导航到**证书**标签
   - 点击 **+**（添加）按钮
   - 选择您的 `.p12` 文件
   - 输入导出时设置的密码
   - 证书将被导入并安全地存储在 Android Keystore 中

### 创建台站

导入证书后，创建台站配置：

1. 导航到**台站**标签
2. 点击 **+**（添加）按钮
3. 填写必填字段：
   - **台站名称**：友好的名称
   - **呼号**：从已导入的证书中选择
   - **DXCC 实体**：从证书自动填充
   - **网格坐标**：您的 Maidenhead 坐标
   - **州/省**：可选，如适用请选择
   - **县/市**：可选，选择州后出现
   - **CQ 区 / ITU 区**：可选
   - **IOTA ID**：可选
4. 点击**保存**

您可以为不同的操作位置创建多个台站。

## 签名 QSO

TaffyQSL 提供两种签名 QSO 日志的方法：

### 方法 1：签名外部 ADIF 文件（日志界面）

使用此方法签名来自其他日志应用程序的 ADIF 文件：

1. 导航到**日志**标签
2. 选择**签名日志**子标签
3. 点击**选择 ADIF 文件**并选择您的 `.adi` 文件
4. 从下拉菜单中选择一个**台站**
5. （可选）设置日期过滤器以仅签名日期范围内的 QSO
6. 点击**签名**
7. 应用将处理文件并显示结果：
   - 总 QSO 数
   - 成功签名
   - 重复项（已在数据库中）
   - 日期过滤
   - 无效记录
8. 签名后：
   - **保存 .tq8**：导出签名文件以供手动上传
   - **上传到 LoTW**：直接上传到 LoTW
   - **关闭**：关闭对话框

### 方法 2：签名内部日志（QSO界面）

使用此方法签名内部日志簿中的 QSO：

1. 导航到**日志**标签 → **ADIF 文件**子标签
2. 点击日志文件以打开它
3. 在 QSO 列表中，点击**上传图标**（右上角）
4. 选择一个**台站**
5. （可选）设置日期过滤器
6. 点击**签名**
7. 按照与方法 1 相同的签名后选项操作

**注意**：两种方法都会检查本地数据库中的重复项，以避免重复签名同一 QSO。

## ADIF 日志簿

TaffyQSL 维护一个内部 ADIF 日志簿来管理您的 QSO。

### 文件管理（日志 → ADIF 文件）

#### 创建新日志文件：
1. 进入**日志**标签 → **ADIF 文件**子标签
2. 点击 **+** 按钮（右下角）
3. 选择**新建 ADIF 日志**
4. 输入名称（例如，"2026 portable ops"）
5. 点击**创建**

#### 导入现有 ADIF 文件：
1. 点击 **+** 按钮
2. 选择**导入 ADIF**
3. 从设备中选择一个 `.adi` 文件
4. 输入显示名称
5. 点击**导入**
6. 应用将显示导入了多少 QSO 以及跳过了多少（重复项）

#### 重命名或删除日志文件：
1. 点击日志文件卡片上的 **⋮**（三个点）
2. 选择**重命名**或**删除**
3. 确认您的操作

### 查看 QSO（QSO 列表屏幕）

点击任何日志文件以查看其 QSO：

- QSO 以卡片形式显示，显示：
  - 呼号
  - 日期/时间
  - 波段、模式、频率等
- 滚动浏览列表
- 点击 QSO 卡片以编辑它
- 使用删除按钮删除 QSO

## QSO 编辑

### 手动 QSO 编辑器（QSO 编辑屏幕）

通过点击 QSO 卡片或**编辑** FAB（小的，右下角）访问：

#### 自动推断功能：
- **频率 → 波段**：当您输入频率时，波段会自动确定
- **卫星 → 频率**：当您选择卫星时，TX/RX 频率和波段会从卫星数据库自动填充
- **手动覆写**：您可以手动编辑任何自动填充的字段；应用将尊重您的更改

#### 保存：
- 点击**保存**以保存更改
- **保存后，表单会重置**以允许快速输入下一个 QSO（对于连续记录多个联系很有用）
- 点击**取消**或返回按钮以返回而不保存

### 快速添加（快速添加屏幕）

通过点击**添加** FAB（大的，右下角）访问：

快速添加允许您使用简化的文本格式输入多个 QSO。每行输入一个 QSO，应用将自动解析信息。

#### 支持的格式：

解析器按以下优先级顺序识别标记：

1. **卫星别名**：常见卫星名称（例如，`SO50`、`AO91`、`ISS`）
2. **模式关键字**：模式名称（例如，`SSB`、`CW`、`FT8`、`FM`）
3. **带单位的频率**：
   - `14074kHz` 或 `14.074MHz` 或 `1.2GHz`
4. **小数频率**：
   - `14.074`（解释为 MHz）
5. **呼号**：业余无线电呼号模式
6. **带分隔符的完整日期**：
   - `2026-03-02` 或 `2026/03/02`
7. **8 位数字日期**：
   - `20260302`
8. **5-6 位数字频率**：
   - `14074`（解释为 kHz）
9. **带分隔符的短日期**：
   - `03-02` 或 `03/02`（假定为当前年份）
10. **时间 (HH:MM)**：
    - `14:30`（UTC）
11. **4 位数字歧义**：
    - 可能是时间（`1430`）、日期（`0302`）或频率（`1230kHz`）
    - 应用将做出最佳猜测并显示**歧义提示**供您澄清

#### 歧义解决：

当解析器遇到歧义的 4 位数字标记时，它会在输入下方显示一个**建议提示**：
- 点击提示以在备选项之间切换（例如，"03:02" → "0302 kHz" → "03-02"）
- 输入文本会**就地修改**并重新解析

#### 示例输入：

```
BG4KNN 14.074 FT8 03-02 14:30
18:15 SO50 BG4KNN BG4LWS BI4KSR BG4JPO
BG4KNN 7074kHz FT8 20260308 0945
```

## LoTW 查询

LoTW 查询功能允许您直接从 ARRL 的 Logbook of The World 检查已确认和已上传的 QSO。

### 设置（首次使用）：

1. 导航到**设置**标签
2. 滚动到 **LoTW 凭据**
3. 点击打开凭据对话框
4. 输入您的 **LoTW 用户名**和**密码**
5. 点击**保存**
6. 凭据将被加密并安全地存储在您的设备上

### 查询 QSO：

1. 导航到 **LoTW** 标签
2. 配置过滤器：
   - **QSL（已确认）/ 已上传**：在已确认的 QSL 和已上传的 QSO 之间切换
   - **波段**：选择特定波段或"任意"
   - **模式**：选择特定模式或"任意"
   - **起始日期**：查询的开始日期
   - **更多过滤器**（点击展开）：
     - **自己的呼号**：按您的呼号过滤
     - **通联呼号**：按您通联的台站过滤
     - **结束日期**：查询的结束日期
     - **显示 DXCC 详情**：包含 DXCC 实体信息（强制"已确认"模式）
3. 点击**查询**
4. 结果以卡片形式显示，显示：
   - 呼号
   - 频率、模式、波段
   - 日期和时间
   - 状态徽章："✓ 已确认"或"已上传"

### 注意事项：

- 查询直接发送到 `https://lotw.arrl.org/lotwuser/lotwreport.adi`
- 结果从 ADIF 格式解析
- 本地不存储数据；每次查询都从 LoTW 获取新数据
- 如果您看到身份验证错误，请在设置中验证您的凭据

## 其他功能

### 设置

通过**设置**标签访问：

- **使用本地时间**：在 UTC 和本地时间之间切换 QSO 时间戳
- **日期格式**：选择您喜欢的日期格式
- **语言**：更改应用语言（Android 13+）
- **LoTW 凭据**：管理您的 LoTW 用户名/密码
- **调试模式**：启用额外的日志记录（仅调试版本）
- **关于**：查看应用版本、源代码和许可证
- **开源许可证**：查看第三方库许可证

### 备份和导出

- **证书备份**：证书存储在 Android Keystore 中，可以导出为加密的 `.p12` 文件
- **ADIF 导出**：导出任何日志文件或签名的 `.tq8` 文件
- **数据库**：QSO 记录存储在本地 SQLite 数据库中
